# ChurnGuard - Subscription Retention Predictor

An end-to-end **Machine Learning + Streamlit web application** that predicts **subscription churn risk** for EdTech users based on their behavioral activity data.

The system helps retention teams identify users who are likely to cancel their subscription early and take proactive actions using personalized offers.

---

## 🚀 Features
- Predict subscription churn risk
- User segmentation: **Safe / At Risk / Critical**
- Churn probability score
- Feature importance visualization
- Random Forest with hyperparameter tuning
- Streamlit interactive web app
- End-to-end ML pipeline
